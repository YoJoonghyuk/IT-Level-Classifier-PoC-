# IT-Level Classifier (PoC)

Данный проект представляет собой Proof of Concept (PoC) системы автоматической классификации уровня квалификации
IT-специалистов (Junior, Middle, Senior) на основе данных резюме с сайта hh.ru.

 ##  Описание проекта
Цель проекта — доказать жизнеспособность идеи автоматического определения грейда разработчика по косвенным признакам:
названию должности, опыту работы, городу и зарплатным ожиданиям.

 ##  Основные этапы пайплайна:

1.  Фильтрация: Из общего датасета выделяются только IT-специалисты по ключевым словам.
2.  Разметка (Labeling): Целевая переменная формируется на основе эвристического анализа (комбинация стажа работы 
и ключевых слов в названии должности).
3.  Feature Engineering: 
    •   Текстовые данные (должность + город) обрабатываются через TF-IDF.
    •   Числовые данные (возраст, опыт, зарплата) масштабируются через StandardScaler.
4.  Классификация: Обучение моделей Logistic Regression и Random Forest с балансировкой весов классов.

## Инструкция по запуску

1. **Подготовка данных**: Положите `hh.csv` в папку `data/`. 

2. **Установка зависимостей**: Установите необходимые библиотеки, используя pip: 

`pip install pandas numpy scikit-learn joblib matplotlib seaborn
`
3. **Парсинг и обработка**: 

`python parse_data.py data/hh.csv`

Результат: в data/ появятся x_data.npy и y_data.npy, в resources/ — обученные скалеры.

4. **Обучение моделей**

`python train_classifier.py`

Результат: в консоль выведется Classification Report, в resources/ сохранятся модели, в docs/ появится график 
баланса классов.

4. **Предсказание**

`python app.py data/x_data.npy
